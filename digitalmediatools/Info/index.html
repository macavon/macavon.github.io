About This Site

This site uses valid XHTML 1.0 and CSS 2, The use of JavaScript has been kept to a minimum; all scripts are based on the W3C DOM objects. The site should display correctly on any browser that correctly implements those standards. The Flash Player plug-in is required to view embedded Flash movies.

We have tested the pages on the following browsers:

  • On Mac OS X: Safari 2.0, Firefox 2.0, Camino 1.0, Opera 9.0 and Lynx 2.8.5.
  • On Windows XP: Internet Explorer 6 and 7 and Firefox 1.5.

There are small cosmetic glitches in IE6 and Opera, which do not interfere with the operation of the site. We urge anybody who is still using IE6, with its numerous bugs and patchy support for standards, to upgrade to either IE7 or Firefox.

We are sorry, but we cannot support older browsers such as Internet Explorer 5 for Mac or Windows, or Netscape 4 or earlier, which do not support Web standards. If you are unable to upgrade to a more modern browser, or prefer not to, we suggest that you disable stylesheets and JavaScript on these old browsers. The main parts of the site should be usable, and the layout will probably be better than the mess these browsers will make attempting to interpret the stylesheet. However, to appreciate the site properly and to experience the examples, you will have to use one of the browsers listed above.

The site was created entirely using the formidable TextMate editor, on a Mac.
