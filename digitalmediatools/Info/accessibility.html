Accessibility

We believe that this site conforms to Level Double-A of the W3C Web Content Accessibility Guidelines, including all Priority 1 and Priority 2 and some Priority 3 checkpoints as defined in the Guidelines.

If you find any violations of these guidelines, or encounter any other accessibility problems, please tell us about it, and we will rectify the matter.

Access Keys

If you have difficulty using a mouse, you can tab between links or use access keys to navigate the site using your keyboard. The following keys are available on all pages where the corresponding links appear.

  1. S: Skip navigation
  2. 1: Home Page
  3. 9: Contact us
  4. 0: Accessibility (this page)

On most Windows browsers you must press the ALT key at the same time as the access key. On MacOS X browsers, you must press the CTL key at the same time as the access key.

Font Size

The site respects your default font size setting. If you cannot read the text easily, you can increase the font size in your browser and the pages will scale up. If you have set a minimum font size which is larger than your default size, however, the layout will be disrupted. We recommend not setting a minimum font size, since this is not the only site on which you will experience problems. Use your browser's commands for changing the font size, or apply your own custom stylesheet instead. If you are using IE7 or Opera, you may prefer to use the Zoom command to blow up the entire page.
