Web Design: A Complete Introduction

Web Design: A Complete Introduction is an original and comprehensive introductory course text intended for anyone who is new to Web design and development. Aimed primarily at college courses, it is used by both computing and design departments as well as in new subject areas such as multimedia design. Its main chapters cover Web experiences, Web technology, markup, stylesheets, Web graphics, time-based media, DOM scripting, Web applications, Web accessibility, Web page design and Web site design. Every aspect of Web design and development is considered, from the technical to issues in visual design, and the book emphasizes the importance of both Web standards and designing for accessibility throughout.

This book is now out of print.
