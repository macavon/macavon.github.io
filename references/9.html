References for Chapter 9

Books

  • Lewis Blackwell, 20th Century Type (Yale University Press: Revised ed., 2004)

    An extremely good history of the important developments in typography in the last century. Anyone who has any interest in fonts and typography should read this book.

Web Pages

  • The Unicode Consortium

    The home page of the Unicode organization, includes FAQs, glossary, new, character tables and the standard itself (in PDF).

  • Adobe Type 1 Font Format

    PDF version of the book “Adobe Type 1
    Font Format” (Addison-Wesley: 1990). This specification provides information on how PostScript Type 1 fonts are stored, and what hints can be provided by the font designer.

  • OpenType Specification

    The official definition of the OpenType font format.
