References for Chapter 15

Books

  • Ian Graham and Liam Quinn, XML Specification Guide (John Wiley & Sons: 1999)

    This book is mostly an annotated version of the specification. For anyone seriously interested in XML, it is essential reading. Unfortunately, it appears to be out of print, but you should be able to find it in a library or get it from a second-hand bookseller.

Web Pages

  • Namespaces in XML

    The short document defining how XML namespaces work.

  • Scalable Vector Graphics (SVG)

    The main page of the W3C’s SVG section, includes links to all the SVG standards, as well as news about SVG and software that supports it.

  • XML Core Working Group Public Page

    This page contains links to all the important XML standards, including the current definition of XML itself. The documents are mostly dense and hard to read. For a definition of XML, you are better off with Graham and Quinn’s book. (See above.)
